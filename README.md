## Features

- No registration needed
- Real-time using web sockets
- Select game duration
- Select player color

## Technologies

### Backend
- NodeJS & Express
- MongoDB
- WebSockets

### Frontend
- React
- React Router
- WebSockets
- Chess.js

## Installation

1. Clone the project

2. Install all dependencies

```bash 
npm install
```

3. Start your MongoDB

4. Rename .env.example to .env and configure environment variables

5. Run server
```bash
npm run start
```
